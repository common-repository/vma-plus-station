=== Vma plus Station Plugin ===

Vma plus Station Plugin は、Wordpress でメタバースが構築できるプラグインです。

== Description ==

Vma plus Station Plugin は、Wordpress でメタバースが構築できるプラグインです。

== Installation ==

プラグインの利用方法を以下に示します。

１．Plugin をインストールいただきます。

２．Wordpress メニューから「固定ページ」を選択し、「新規固定ページを追加」で

　　ページを作成します。

３．固定ページに「Metaverse」のテンプレートを選択する。

４．固定ページを保存する。

５．パーマリンクＵＲＬより、あなただけのメタバースが表示できます。
