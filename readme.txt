=== Description ===

Vma plus Station Plugin is a plugin that allows you to easily build a metaverse on WordPress. It enables users to initiate a virtual space and offers features such as free movement within the virtual environment and the option to set destination URLs as thumbnails.

== Installation ==

**Installation from within WordPress**

 - Visit **Plugins > Add New**.
 - Search for **Vma plus Station**.
 - Install and activate the **Vma plus Station** plugin.

<br/>

**Manual installation**

 - Upload the entire `vma-plus` folder to the `/wp-content/plugins/`ã€€directory.
 - Visit **Plugins**.
